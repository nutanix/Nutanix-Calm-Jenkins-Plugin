Select the Nutanix Calm credentials.

To add a new Nutanix Calm credential click on Add. Select Jenkins.

In Jenkins Credentials Provider

Retain the default values in Domain, Kind and Scope fields.

Username is mandatory. Enter the Prism Central user authorized to use Calm.

Password is mandatory. Enter the password for the Prism Central user authorized to use Calm.

ID and Description fields are optional. Provide the description to uniquely identify the credential if needed.
